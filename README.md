# Minesweeper
Console game minesweeper

## Controls
| Key   	    | Action          |
|---	        |---	            |
| WASD   	    | movement        |
| O [letter]  | set/unset flag  |
| Enter  	    | open cell      	|
| Esc  	      | back to menu  	|
